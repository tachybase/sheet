# 如何协作编写文档

## 你需要安装 `pnpm` 环境

首先你需要安装 [pnpm](https://pnpm.io/zh/installation) 的环境

```bash
  git clone https://github.com/tachybase/sheet
  # 初始化
  pnpm install
  # 项目启动
  pnpm doc:dev
  # 然后开始愉快的开始协作吧~
```
